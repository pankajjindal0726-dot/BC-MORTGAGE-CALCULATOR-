## ğŸ¯ 2025 BC Mortgage Rules

This calculator implements:
- Minimum down payments (5% on first $500k, 10% on $500k-$1.5M, 20% above $1.5M)
- CMHC insurance premiums (0.6% to 4.5% based on LTV)
- Property Transfer Tax with exemptions
- 30-year amortization for first-time buyers and new builds
- Mortgage stress test (rate + 2%)
- First-time buyer PTT exemption (up to $835k full, $860k partial)
- Newly built home PTT exemption (up to $1.1M full, $1.15M partial)
